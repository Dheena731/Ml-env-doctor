# CLI reference

## `mlenvdoctor diagnose`

```bash
mlenvdoctor diagnose [--full] [--json PATH|-] [--csv PATH] [--html PATH]
```

## `mlenvdoctor doctor`

```bash
mlenvdoctor doctor [--ci] [--guided] [--json] [--full]
```

## `mlenvdoctor report`

```bash
mlenvdoctor report [--quick|--full] [--output-dir PATH]
```

## `mlenvdoctor fix`

```bash
mlenvdoctor fix [--conda] [--venv] [--stack STACK] [--plan] [--dry-run] [--apply] [--yes] [--verify] [--rollback]
```

## `mlenvdoctor stack llm-training`

```bash
mlenvdoctor stack llm-training [--output PATH]
```

## `mlenvdoctor dockerize`

```bash
mlenvdoctor dockerize [MODEL] [--service] [--stack STACK] [--base-image IMAGE] [--python-version VER] [--output FILE]
```

## `mlenvdoctor mcp serve`

```bash
mlenvdoctor mcp serve
```
